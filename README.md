# 🎓 CampusPrep AI

An AI-powered learning platform designed to help college students prepare for campus examinations from the **1st year to the 4th year**. CampusPrep AI provides personalized study plans, AI-generated notes, quizzes, previous year paper analysis, and progress tracking to make exam preparation smarter and more efficient.

---

## 🚀 Features

* 📚 AI-powered study planner
* 🤖 AI tutor for doubt solving
* 📝 Smart notes generation
* 📄 Previous year paper analysis
* 🎯 Personalized learning roadmap
* 📊 Student progress tracking
* ⏰ Smart revision timetable
* 📋 Mock tests and quizzes
* 💡 Flashcards for quick revision
* 🔐 Secure authentication using JWT
* 📱 Responsive UI for desktop and mobile

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/isha282006/CamousPrepAI.git
```

### Install dependencies

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=Your_MongoDB_URI

JWT_SECRET=Your_JWT_Secret

GEMINI_API_KEY=Your_Gemini_API_Key
```
